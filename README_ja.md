# 自然界のすべては循環でできている

> English version: [README.md](./README.md)

> 本リポジトリは、自然界の生命維持システムが「直線的な因果」ではなく「循環」によって成り立っていることを、基礎科学・生態学・地球科学の観点から整理した日本語版です。自然補完科学の基礎となる「循環」の考え方を、わかりやすく説明する入門文書です。

## 概要

人類は、自然の構造を根本的に誤解してきました。

地球上の生命は、直線的な消費ではなく、循環によって維持されています。

これは中学校の理科や生物でも扱われる基礎的な内容ですが、現代社会、そして時に科学者でさえ、その意味を十分に文明設計へ反映できていません。

この文書では、地球の生命維持システムがどのように機能し、なぜその循環を壊すことが人類自身を含むすべての生物に危機をもたらすのかを整理します。

## 1. 水循環：生命の基盤

もっとも単純で、もっとも重要な自然循環が水循環です。

```text
海の水が蒸発する
→ 雲になる
→ 雨として降る
→ 川を流れる
→ 海へ戻る
```

山と森林はこの循環に大きな役割を持ちます。

- 木々は水蒸気を放出する。
- 森林は雲を作る。
- 雨は土地を潤す。

この全体は、直線ではなく閉じた循環です。

## 2. 土壌・雨・栄養塩の流れ

雨は、単に地面を濡らすだけではありません。

雨は以下を運びます。

- ミネラル
- 有機物
- 分解された落ち葉
- 腐植・腐葉土

この栄養豊かな流れは、山から川へ、川から湖へ、そして海へと移動します。

これは地球の栄養搬送ベルトです。

この流れが止まれば、生態系の生産性は維持できません。

## 3. 海洋食物網：微小生命が支える生命ピラミッド

陸から運ばれた栄養塩は、海の生命を支えます。

### 植物プランクトン

植物プランクトンは、陸上植物と同じように光合成を行います。地球の酸素の大きな割合を支える重要な存在です。

### 動物プランクトン

動物プランクトンは植物プランクトンを食べます。

### 魚類・海洋生物

小魚、中型魚、大型魚へと食物連鎖が続きます。

### 海鳥・海獣・人間

ペンギン、アザラシ、クジラ、そして人間も、この循環の上に存在しています。

さらに、生物の排泄物や死骸は微生物によって分解され、栄養として再び水中へ戻ります。

これが生命の循環です。

## 4. 多様性は選択肢ではなく循環のエンジンである

単一の種だけでは、循環は維持できません。

循環には以下が必要です。

- 生産者
- 消費者
- 分解者
- 捕食者
- 腐食者

それぞれの種が異なる役割を持っています。

一つの重要な役割が失われるだけでも、全体の循環は崩れます。

```text
多様性 = 持続性
循環停止 = 生命停止
```

## 5. 自然循環は普遍的な自然法則に従う

自然循環は偶然ではありません。

それは以下の普遍的な原理に従っています。

- 循環
- 均衡
- 調和
- フィードバック
- 再生

これは、自然法則、または持続的存在の法則です。

長期的に存続するものは、必ずこの原理に従っています。

## 6. 二元論的思考は循環を壊す

現代文明はしばしば、以下のような二元論で物事を考えます。

- 勝ち / 負け
- 善 / 悪
- 使用 / 廃棄
- 利益 / 損失
- 人間 / 自然

しかし自然は、直線的でも、二項対立でも、一方向でもありません。

自然は、循環的で、関係的で、再生的です。

人間が循環を壊す行動を取れば、以下が減少します。

- 生物多様性
- 土壌肥沃度
- 水質
- 海洋生産性

最終的には、人類自身の生存可能性を低下させます。

## 7. まとめ：すべての生命は循環であり、人類もその一部である

人類が依存しているものは、すべてこの循環から来ています。

- 呼吸する空気
- 飲む水
- 食べる食料
- 生態系の安定

このシステムのどこかを壊せば、全体が不安定化します。

循環を理解することは、高度な科学ではありません。基礎生物学、基礎生態学、基礎地球科学です。

しかし現代文明は、まるでこの法則が自分たちには適用されないかのように振る舞ってきました。

自然は交渉できません。

循環は、人類が従っても従わなくても続きます。

人類は、その循環の中に残る道を選ぶのか、それとも外される道を選ぶのかを問われています。
